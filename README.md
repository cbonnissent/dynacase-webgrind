# dynacase-webgrind

Integrates [webgrind](https://github.com/jokkedk/webgrind) (Xdebug Profiling
Web Frontend in PHP) as a Dynacase's admin service.

See : http://www.xdebug.org/docs/profiler#starting to launch the profiling
